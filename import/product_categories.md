## Fallback / Priority Rules for Claude Code

When matching products from `products.json` to categories, apply in this order:

1. **Series code match** — check if `name` starts with or contains a known series code (e.g. `DT1`, `BBH3`, `X90`). Series codes are the most reliable identifier.
2. **Parent category keyword** — if no series match, use keywords like `Sägeblatt`, `Bohrer`, `Spannfutter`, `Fräser`, `Messer`.
3. **Fallback** — assign to the closest parent category and flag for manual review.

### Ambiguous Series (need context)
- `FVI` appears in both **VHW Fräser** (Abrundprofil) and **Verbundwerkstoffe** (Anfasen) — check product description
- `FDT` / `FDH` appears in both **DP & HW** and **Verbundwerkstoffe** — check if name contains `Verbund` or `PVC` or `Aluminium`
- `PKA` appears in both **Kantenanleimmaschinen** and **Kreissägeblätter** — check if name contains `Sägeblatt` or `Fräskopf`
- `DTR` is listed under **Verbundwerkstoffe** (p.128), not the regular DT series

## slug generation rule (reuse from INSTRUCTIONS.md)
```python
import re
def slugify(name):
    s = name.lower()
    s = s.replace('ä','ae').replace('ö','oe').replace('ü','ue').replace('ß','ss')
    s = re.sub(r'[^a-z0-9]+', '-', s)
    s = s.strip('-')
    return s
```
